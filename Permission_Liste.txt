-------- Permissions --------

******** Lobby *******

world.*: Permission die Welt zu verändern
  world.destroy: Permssion Blöcke abzubauen
  world.place: Permission Blöcke zu bauen

command.*: Permssions für alle Commands
  command.1vs1: Permission für den 1vs1-Command
  command.refuse: Permission für den refuse-Command
  command.team: ...
  command.setminkit: ...
  command.setmaxkit: ...
  command.warteschlange: ...
  command.turnier.* Permission für alle Tunrier-Command spezifischen Dinge
    command.tunrier.create: ...
    command.turnier.join: ...
  command.setdifkit: ...
  command.setprekit: ...
  command.ffa: ...
  command.kit: ...
  command.spectate: ...
  command.stats:*: Permission für alle Stats anschauen/resetten
    command.view: Permission Statistiken anzuschauen
    command.reset: Permission seine Statistiken zurückzusetzen
    command.resetother: Permission Statsitiken eines anderen Spielers
    zurückzusetzen
  command.queueinfo: ...
  command.rankedqueueinfo: ...
  command.forcequeue: ...


******** Arena *******

command_arena.*: Permission für alle Arena-Commands
  command_arena.setwaitinghouse: Permission für den setwaitinhouse-Command
  command_arena.giveup: ...
  command_arena.fix: ...
  command_arena.endmatch: ...
  command_arena.stats: ...
  command_arena.turnier: ...

no_push_back: Permission, die bestimmt, ob man als Zuschauer nicht zurück gestoßen wird


******* Setup *******

world.* Permission die Welt zu verändern (außerhalb von Kämpfen)
  world.place: Permission Blöcke zu platzieren
  world.destroy: Permission Blöcke zu zerstören

command_setup.*: Permission für die Setup-Commands
  command_setup.spawn: Permission für den spawn-Command
  command_setup.setpsawn: ...
  command_setup.map: ...
